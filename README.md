# 10FaktorApp

Prosjektet er avhengig av to filer; config.ini og 1_Data.xml. Sistnevnte er et uttrekk med persondata hentet ut fra Vismas endepunkt for persondata. Hele xml-en kan benyttes. Dette programmet er låst til å sammenligne data fra xml-fila med Eadm-brukere fra Fredrikstad kommune. Hvis Hvaler skal hentes ut, må EadmManager-fila redigeres slik at Hvaler legges inn som organisasjon i uttrekket fra Eadm.

Config.ini er ikke lagt ved, da denne fila inneholder sensitive nøkler. Fila er lagret på vårt Teams-område under dokumentasjon/Programvare/10-Faktor.
